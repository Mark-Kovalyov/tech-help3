# Erlang language

## Data types

* Terms
* Number
  * $char
  * base#value
* atom
  * an atom is a literal, a constant with name
* bit strings and binaries
* reference
* fun (functional ref)
* port identifier
* PID
* tuple
* map
* list
* string
  "string" "42" == "string42"
* record
* boolean
  * there are no boolean, only atoms true|false used.

## Conditions

```
if
  cond1 -> Action1;
  cond2 -> Action2;
end
```
