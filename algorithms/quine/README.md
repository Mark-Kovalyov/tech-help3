# Quine and Quine-McCluskey

## De-Morgan
```
!(a * b) == !a + !b
!(a + b) == !a * !b
```

## Операция склеивания

```
w * x + w * !x == w * (x + !x) == w
```

## Операция поглощения (Annihilator)

```
w + w * z == w * (1 + z) == w
```
